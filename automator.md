    - 단축키 입력 시 iterm이 실행된다.

5. ctrl + cmd + V 키로 Visual Studio Code 열기

11. ctrl + cmd + Z 키로 Zeplin 열기

    - 단축키 입력 시 사이트가 열린다.

    - 단축키 입력 시 mobile의 깃헙 페이지가 크롬으로 열린다.

15. ctrl + cmd + E 키로 펫트너 petsitter의 깃헙 페이지 열기
    - 단축키 입력 시 petsitter의 깃헙 페이지가 크롬으로 열린다.
