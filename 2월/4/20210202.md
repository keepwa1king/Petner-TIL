2. forEach 함수
    - 내부 반복을 수행하는 함수
    - 외부에서 봤을 때는 반복문 형태를 띄지 않지만 내부적으로는 반복을 하고 있음
    ```
    items = [1, 2, 3, 4, 5];
    items.forEach(print); // 1, 2, 3, 4, 5
    ```
